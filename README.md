# Panda Express Platformer Game

## Overview
Panda Express Platformer Game is a browser-based side-scrolling platformer built using JavaScript and the p5.js library. The project focuses on interactive gameplay mechanics, modular game design, collision systems, and responsive player controls.

The game was developed to demonstrate programming fundamentals, object-oriented thinking, debugging practices, and interactive front-end development concepts.

## Features

- Side-scrolling game world
- Responsive player movement and jumping mechanics
- Collectible coins system
- Canyon collision and fall detection
- Lives and scoring system
- Game over and restart states
- Enemy interactions
- Platform mechanics
- Sound effects integration
- Animated game elements
- Modular and reusable game object logic

## Technologies Used

- JavaScript (ES6)
- p5.js
- HTML5
- CSS3

## Programming Concepts Demonstrated

### Object-Oriented Programming
